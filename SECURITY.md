# Security Policy

## Supported Versions

Currently, only the `main` branch of Stadium AI Copilot is supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| main    | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

Security is a top priority for Stadium AI Copilot.

If you discover a security vulnerability within this project, please **do not** disclose it publicly via GitHub Issues.

Instead, please send an email directly to the maintainers at [INSERT EMAIL]. We aim to acknowledge receipt of the vulnerability report within 48 hours and provide a timeline for resolution.

### What to Include in Your Report

To help us quickly understand and reproduce the issue, please include:

- A detailed description of the vulnerability.
- Steps to reproduce the issue.
- Potential impact and risk assessment.
- Any relevant context regarding the deployment environment.

We deeply appreciate your help in keeping Stadium AI Copilot secure and safe for all users!
